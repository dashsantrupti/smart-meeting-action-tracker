## 🌐 Smart Meeting Action Tracker


🧾 Overview

Smart Meeting Action Tracker is a full-stack web application that helps users convert meeting discussions into structured, actionable tasks. The application streamlines meeting management by organizing notes, assigning responsibilities, tracking deadlines, and monitoring task progress through an interactive dashboard. It also integrates AI to automatically generate action items from meeting notes.

---
🎯 Objective

To build a scalable full-stack application that demonstrates:
- Full-stack web development using React and Flask
- RESTful API development and backend architecture
- User authentication and secure access control
- Relational database design using MySQL
- AI integration for automated task generation
- Real-world task and meeting management workflows

---
⚙️ Features
- User registration and secure authentication
- Create, update, delete, and manage meetings
- AI-powered task generation from meeting notes
- Task assignment, priority management, and deadline tracking
- Search and filter meetings and tasks
- Interactive dashboard with task progress and analytics
- Responsive user interface for desktop and mobile devices

▶️ How to Run

```bash
git clone https://github.com/dashsantrupti/smart-meeting-action-tracker.git

cd smart-meeting-action-tracker
```

### Backend

```bash
cd backend

pip install -r requirements.txt

python app.py
```

### Frontend

```bash
cd frontend

npm install

npm start
```

Configure the MySQL database and environment variables before running the application.
